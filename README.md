# ToDoList App with Python, Flask & Supabase

A simple **To-Do List web application** built with **Python, Flask, JS, HTML**, and **Supabase**.  
Demonstrates full CRUD functionality: add, edit, delete, and mark tasks as done.

> **Note:** The Supabase database is **not included**. You need to create your own project to use the app.

---

## Features

● Add, edit, delete tasks  
● Mark tasks as done/undone  
● Full CRUD using Flask & Supabase

---

## Tech Stack

● Python  
● Flask  
● Supabase (PostgreSQL + REST API)  
● HTML, CSS, JS

---

## Supabase Table Setup

Use the following SQL command in your Supabase project to create the `todos` table:

```sql
create table todos (
  id bigint generated by default as identity primary key,
  title text not null,
  done boolean default false,
  inserted_at timestamptz default now()
);
